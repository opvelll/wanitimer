# wanitimer

死まであと何日か表示する Web アプリ. 元ネタ:100 日後に死ぬワニ. AWS Amplify Sample

blog: https://opvel.com/2020/03/24/post-1061/

site: https://wanitimer.d2v7szbbzh7lnj.amplifyapp.com/ (たぶんそのうち消す)

![wanitimer](./src/assets/wanitimer3.png)

## Project setup

```
git clone wanitimer
cd ./wanitimer

npm install

amplify init

amplify push

npm run serve
```
